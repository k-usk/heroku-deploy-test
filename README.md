# heroku-deploy-test
Githubのリポジトリにプッシュしたらherokuに自動でデプロイされるかどうかのテストをテストしてみた。

##結果


##特定のブランチのみを対象にできるか？
できる。  
自動でデプロイするブランチを選択出来る。  
また、特定のブランチを選んで手動でのデプロイも可能。
